# Biblioteca (Flask + SQLite)

## Como rodar
1. Crie e ative um virtualenv (opcional).
2. Instale dependências:
   ```bash
   pip install flask
   ```
3. Rode a aplicação:
   ```bash
   python app.py
   ```
4. Acesse http://127.0.0.1:5000 no navegador.

## Dicas
- Use `/seed` para inserir dados de exemplo rapidamente.
- Filtro por categoria disponível na página "Listar".
- Banco de dados: `biblioteca.db` (SQLite).
